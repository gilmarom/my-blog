Let's style with style

with angular/flex-layout we'll design a responsive website with minimum time and effort, reduce css and gain control on our code

Angular Flex Layout provides a sophisticated layout API using FlexBox CSS + mediaQuery. The Flex Layout engine intelligently automates the process of applying appropriate Flexbox CSS to browser view hierarchies. This automation also addresses many of the complexities and workarounds encountered with the traditional, manual, CSS-only application of box CSS.
